# discord_bot
## To install and run:
1. Clone the repo
   ```sh
    git clone https://github.com/Namanskshetty/discord_bot.git
    ```
2. Go to folder
   ```sh
   cd discord_bot
   ```
3. Give the executable permission to bin.sh file
   ```sh
   chmod +x bin.sh
   ```
4. Run the bash file
    ```sh
    ./bin.sh
    ```
